Features

Supports Hessian 1.0.2 Encoding and Decoding of the following

  • Strings
  • Integers (32 and 64 bit)
  • Dates
  • Floating point numbers
  • Booleans
  • Null values
  • Lists (untyped)
  • Maps (untyped and typed)
  • Calls
  • Replies (void return types and with arguments)
  • Faults (Service or applications exceptions)

Integration scenarios

  • Integration scenario with a remote Java Hessian server
